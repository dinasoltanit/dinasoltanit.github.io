Course Description

MATLAB is a programming language developed by MathWorks that is used for numerical computing tasks including matrix manipulations, data visualizations, and advanced mathematical computations. In this introductory course, we are going to prepare undergraduate students to learn how to code in MATLAB, how to use its toolboxes, and in general, how to implement algorithms with this programming environment.

Syllabus

  • MATLAB Installation and Environments (Scripting, Live Scripting, Code in Command Window)
  • MATLAB Basics
  • Simple Algorithms in MATLAB
  • Variables and Identifiers
  • Statements
  • Operators
  • Conditionals
  • Loops
  • Functions
  • 2D, 3D Plot Strategies
  • Working with Files
  • Working with MATLAB Packages
  • Working on exempted questions
